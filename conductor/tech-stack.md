# Core Technologies\n\n**Mindful** is built with modern, high-performance, and privacy-focused technologies.\n\n## Programming Language\n- **Dart:** High-performance and versatile, perfect for building beautiful and efficient mobile apps.\n\n## Frameworks\n- **Flutter:** Multi-platform UI framework for building beautiful, native, and fast apps for Android.\n\n## Backend & Storage\n- **Drift (SQLite):** Powerful and type-safe database for Android apps, providing a robust and reliable storage layer.\n- **Local-First Architecture:** No data is stored off-device, ensuring user privacy and security.\n\n## State Management\n- **Riverpod:** A reactive and safe state management library for Flutter, providing a clean and maintainable architecture.\n\n## UI/UX Libraries\n- **fl_chart:** For data visualization of screen time and app usage.\n- **flutter_animate:** For smooth and interactive animations.\n- **dynamic_color:** Support for Android's Material You dynamic colors.\n- **fluentui_system_icons:** A consistent set of icons for a clean and professional look.\n\n## Core Services\n- **Local Authentication:** Uses `local_auth` to secure app access with biometrics.
